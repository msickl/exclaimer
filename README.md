# Exclaimer Installation

## Install

Clone the repository into `/opt/exclaimer`:

```bash
git clone https://github.com/msickl/exclaimer.git /opt/exclaimer
```

Change into the configuration directory:

```bash
cd /opt/exclaimer/etc
```

Run the installer:

```bash
python3 install.py
```

## Requirements

* Python 3
* Postgres
* Git
* Linux system with write access to `/opt/exclaimer`

## Notes

Run the installer with sufficient permissions if required (for example using `sudo`).
